# Meet Toba

Your tiny AI for everyday wins.

Toba turns a one-minute voice check-in into three realistic lifestyle actions. The demo includes:

- Toba calling flow with Nokia ringtone and local voice clips
- Alex / they-them persona and gentle-momentum energy plan
- Stardust rewards linked to social, creative, movement, nature and family activities
- Family Circle and Clinician Dashboard views
- Mendra-inspired editorial landing page with Toba's original color palette

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173` (or the port shown by Vite).

This is a front-end hackathon prototype. Voice playback, health signals, clinical summaries and reward redemption are simulated for demonstration purposes.
